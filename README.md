A [Flarum](http://flarum.org) extension. 

## Installation

Install with composer:

```sh
composer require lewuocvi/knoxextchecker:"*"
```

## Updating

```sh
composer update lewuocvi/knoxextchecker:"*"
php flarum migrate
php flarum cache:clear
```
